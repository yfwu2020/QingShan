# 《青山道》关键情节记录
（按章节顺序记录，每完成一章更新一次）

| 章节 | 关键情节 | 影响 |
|------|----------|------|
| 第01章 | 季度小比上张浩然用大道理逼迫王青山退门，苏清出手打伤张浩然护着王青山，玄阳子表面罚苏清实则给王青山送疗伤药，张浩然怀恨在心 | 张浩然与王青山、苏清的矛盾彻底公开，张浩然决定找叔父帮忙报复 |
| 第02章 | 张浩然找叔父张长老问罪，要废掉王青山修为，玄阳子曝光张长老偷拿门派丹药给私生子的黑料，罚张长老闭关三年、张浩然扫半年茅厕，给了王青山十颗聚气丹、苏清上品法器，王青山当晚突破炼气二层，姐姐留下的草蚂蚱吊坠散出金光 | 张明德叔侄不敢再明着找麻烦，王青山修炼速度异常提升，姐姐留下的吊坠伏笔埋下 |
| 第03章 | 张浩然在后山放引兽香引来疾风狼想害王青山，王青山靠吊坠预警躲过致命攻击，引狼到苏清修炼处反杀，玄阳子抓张浩然现行罚他去矿洞挖三个月矿，王青山炼化疾风狼内丹三天突破到炼气三层，千里之外玄元宗的姐姐感应到王青山的气息 | 张浩然暂时无法作恶，王青山修炼进度再提升，姐弟连心的伏笔确认，姐姐所在的玄元宗线索出现 |
| 第04章 | 玄阳子宣布半年后选拔3名弟子下山历练，优秀者奖励筑基丹，所有人不看好王青山，王青山半年内靠努力和吊坠帮助突破到炼气七层，考核时爆冷拿到第三个历练名额，与苏清、张浩然一同三天后出发去黑石城，而黑石城正被青玄宗弟子打着整肃风纪的旗号祸害 | 王青山用实力打脸嘲讽者，获得下山寻亲的机会，即将与伪善的青玄宗势力发生冲突 |
| 第05章 | 三人抵达黑石城，撞见青玄宗弟子收过路费欺负老汉，张浩然假意劝阻王青山实则想让他送死，王青山和苏清出手打跑青玄宗弟子，从百姓口中得知青玄宗在这里作恶多端，张浩然暗中投靠青玄宗告密，王青山从老汉口中得知姐姐可能被青玄宗抓走，青玄宗筑基期队长刘虎准备带人报复 | 王青山与青玄宗矛盾公开，张浩然彻底倒向反派，姐姐下落的线索指向青玄宗，更大的危机即将到来 |
| 第06章 | 刘虎带人找上门围堵，张浩然公开背刺投靠青玄宗，苏清不敌筑基期刘虎受伤，危急时刻草蚂蚱吊坠爆发力量让王青山临时突破到炼气九层，两人联手打伤刘虎，救出被青玄宗关押的姑娘，从获救姑娘口中确认姐姐王秀莲被抓到青玄宗青云山总坛，两人带着百姓躲入城外大山，刘虎全城悬赏搜捕两人 | 确认了姐姐的下落，吊坠的特殊能力初步显现，张浩然彻底成为反派走狗，主线目标更新为救姐姐 |
